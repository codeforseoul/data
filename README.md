데이터(raw data)
================
코드포서울에서 가공하거나 크롤링한 모든 데이터셋을 모으는 레포지토리입니다.

This repository includes all datasets we wrangle or crawl at Code for Seoul.

목록(List of Datasets)
----------------------

#### [정치(politics)](politics)

* 19대 국회의원 프로필
* 19대 국회의원 본회의/상임위원회 출석 정보
* 19대 국회의원 표결 현황

#### [경제(economy)](economy)

* 3년 만기 대한민국 정부채권 수익률
